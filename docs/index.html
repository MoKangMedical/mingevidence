核心功能

覆盖循证医学研究全流程的专业分析工具

文献质量评估

基于 AMSTAR-2、ROBIS 等国际标准工具，AI 辅助完成文献质量的快速评估与分级，减少人工评分的主观偏差。

证据等级分类

遵循牛津循证医学中心证据等级体系，自动对纳入研究进行 1-5 级证据分类，支持自定义分类规则。

GRADE 评价

内置完整的 GRADE 证据质量评价流程，涵盖偏倚风险、不一致性、间接性、不精确性和发表偏倚五个维度。

偏倚风险评估

支持 RoB 2.0、ROBINS-I 等主流偏倚风险评估工具，自动生成偏倚风险总结图和流量图，直观呈现评估结果。

森林图生成

一键生成符合出版标准的森林图，支持固定效应和随机效应模型、亚组分析、敏感性分析等多种统计图表。

报告自动化

基于 PRISMA 2020 规范自动生成系统评价报告框架，支持 Word、LaTeX 多种导出格式，适配 50+ 主流期刊模板。
